# Factory Optimization System

## 📌 Overview
This project is a data-driven decision support system that recommends the best factory for product delivery using machine learning, simulation, and optimization.

## ⚙️ Features
- Predict delivery lead time
- Simulate multiple factory assignments
- Recommend optimal factory
- Interactive dashboard (Streamlit)
- Visualization (charts, tables, map)

## 🧠 Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Plotly
- Pydeck

## 📊 Output
- Best factory recommendation
- Lead time prediction
- Comparison chart
- Map visualization

## ▶️ How to Run
pip install -r requirements.txt
python -m streamlit run app/streamlit_app.py
